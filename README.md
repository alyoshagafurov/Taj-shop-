# 🛍 TAJSHOP — Интернет-магазин мужской одежды

> **Душанбе, Таджикистан** · [@magazin1.nav](https://instagram.com/magazin1.nav) · 356K+ подписчиков

Полнофункциональный демо интернет-магазин для **TAJSHOP** — популярного магазина мужской одежды, обуви и аксессуаров в Душанбе. Работает как статический сайт с клиентским хранилищем (localStorage) — без бэкенда.

---

## ✨ Возможности

### Покупателям
- **Каталог** — 24 товара в 5 категориях с реальными фото (60 снимков)
- **Карточка товара** — выбор цвета (фото меняется), размера, рейтинг и отзывы
- **Корзина** — добавление, изменение количества, оформление заказа
- **Акции** — скидки с таймером обратного отсчёта
- **Бронь** — забронировать товар для примерки в магазине
- **Избранное** — сохранить понравившееся
- **Личный кабинет** — история заказов, избранное, брони, профиль
- **Поиск и фильтры** — по категориям, цене, названию
- **Адаптивный дизайн** — мобильные, планшеты, десктоп

### Владельцам (админ-панель)
- **Дашборд** — статистика: товары, заказы, выручка, клиенты
- **Управление товарами** — добавить, изменить, удалить; цвета с фото
- **Заказы** — просмотр и смена статуса (новый → подтверждён → доставлен)
- **Акции** — временные скидки с указанием % и срока
- **Клиенты** — список зарегистрированных покупателей

## 📁 Структура проекта

```
taj-shop/
├── index.html        # Лендинг — продаёт доверие и желание
├── catalog.html      # Каталог — товары с фильтрами и поиском
├── admin.html        # Админ-панель (пароль: admin)
├── styles.css        # Стили магазина
├── admin.css         # Стили админки
├── data.js           # Стартовый каталог (seed-данные)
├── store.js          # Слой данных — «мини-бэкенд» на localStorage
├── app.js            # Логика магазина
├── admin.js          # Логика админки
├── images/           # 60 фото товаров (p01.jpg – p60.jpg)
├── server.js         # Express-сервер для деплоя
├── package.json      # Node.js зависимости
└── Dockerfile        # Docker-образ для Railway
```

## 🚀 Запуск

### Локально (без установки)
```bash
# Просто откройте index.html в браузере
open index.html
```

### Через Python
```bash
python3 -m http.server 5599
# Магазин:  http://localhost:5599/
# Админка:  http://localhost:5599/admin.html
```

### Через Node.js (как на сервере)
```bash
npm install
npm start
# Магазин:  http://localhost:3000/
# Админка:  http://localhost:3000/admin.html
```

## 🌐 Деплой на Railway

1. Нажмите **Deploy on Railway** или создайте проект вручную
2. Подключите этот репозиторий
3. Railway автоматически обнаружит `Dockerfile` и задеплоит
4. Получите публичную ссылку — можно вставить в Instagram bio

Переменные окружения:
| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `PORT` | `3000` | Порт сервера (Railway задаёт автоматически) |

## 🏗 Архитектура

```
Лендинг (index.html)          Каталог (catalog.html)
     │                              │
     │  «Смотреть каталог»          │  ?cat=obuv&sale=1&q=поло
     └──────────────────────────────►│
                                    │
        ┌───────────────────────────┘
        ▼
   ┌─────────┐    ┌──────────┐    ┌──────────┐
   │ data.js  │───►│ store.js │◄───│ admin.js │
   │ (seed)   │    │ (CRUD)   │    │ (панель) │
   └─────────┘    └────┬─────┘    └──────────┘
                       │
                  localStorage
                  (синхронизация
                  между вкладками)
```

- **Лендинг** → разогревает: hero, категории, хиты, отзывы, контакты
- **Каталог** → продаёт: сетка товаров, фильтры, поиск, сортировка
- **Админка** → управляет: CRUD товаров, заказы, акции, клиенты
- **store.js** → единый слой данных, синхронизация через `storage` event
